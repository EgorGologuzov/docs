# Отладка по  Wi-Fi в Android Studio

## Первоначальная настройка

1. Подключить ПК и телефон к одной сети Wi-Fi
2. В параметрах разработчика телефона включить отладку по Wi-Fi, провалиться в настройку, открыть сканер QR-кодов
3. В Android Studio: Device Manager > Pair Devices Using Wi-Fi
4. Отсканировать QR-код
5. Если ошибка подключения или долгое подключение:
	- попробовать отключить VPN / подключить ПК и телефон к одному серверу
	- перезапустить Android Studio и Wi-Fi на телефоне
6. Если соединение сбрасывается после выключения экрана телефона:
	- в настройках телефона: Подключения > Wi-Fi > ... > Intelligent Wi-Fi > Автоподключение к точке доступа > Отключать, если не используется = Выключить
	- отключить режим энергосбережения
