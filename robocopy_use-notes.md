# Заметки по использованию утилиты robocopy (Windows)

Команда для отзеркаливания файлов:
```cmd
robocopy "C:\Src" "D:\Dest" /MIR /R:2 /W:5 /TEE /UNILOG:"C:\Src\sync.log" /NJH /NJS
```
- Копирует все новые файлы, заменяет старые версии на новые и удаляет все лишние файлы (которое есть в Dest, но нет в Src)
- Пишет лог в консоль и файл C:\Src\sync.log

### Копирование проектов с большим кол-вом мелких файлов

Добавить к команде параметр /XD к именами папок для игонирования, например папки .bin .cache:
```cmd
robocopy "C:\Src" "D:\Dest" /MIR /R:2 /W:5 /TEE /UNILOG:"C:\Src\sync.log" /NJH /NJS /XD .bin .cache
```

Для проектов на Node.js после копирвания нужно восстановить папку .bin командой:
```cmd
npm rebuild
```
После этого проект должен корректно запуститься
