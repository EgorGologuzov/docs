# Установка Windows 10

- запуститься с флешки и выполнить стандартную установку
- отключить обновления
    - использовать утилиту на рабочем столе Utilites/windows-updates-pause
    > или взять ее отсюда: https://drive.google.com/drive/folders/1wQaF4INcJ8OR8-AMm5nw3raGGw4A6bG3?usp=drive_link
- активировать
    > видео: https://drive.google.com/file/d/1d5dZoNHumPD2AIn9v9SpQn2Z_Hvwta4E/view?usp=drive_link
    - открыть Power Shell от админа
    - irm https://get.activated.win | iex
    > загрузка может быть очень долгой, если нет сообщения об ошибке - просто ждать
    - нажать 1 (опция 1)
    - дождаться завершения процесса
- ДЛЯ ВИРТУАЛКИ: установить дополнения гостевой ОС
    > нужно для буфера обмена и передачи файлов
    - окно ВМ: Устройсва > Подключить образ дополнений гостевой ОС
    - внутри ВМ: этот ПК > виртуальный диск VirtualBox
    - запустить VBoxWindowsAdditions.exe
    - установить с параметрами по умолчанию
    - перезагрузить ВМ
    - включить общий буфер, окно ВМ: Устройства > Общий буфер > Двунаправленный
    - drag and drop, окно ВМ: Устройства > Функция Drag & Drop > Двунаправленный
    - общие папки при необходимости, окно ВМ: Устройства > Общие папки
