# Способы отправки http запросов для VS Code

## Расширение REST Client

- Это расширение для позволяет писать HTTP запросы как код и отправлять их без ограничений
- Чтобы пользоваться нужно установить расширение `REST Client` от `Huachao Mao`
- Примеры написания запросов можно найти в проекте iQmenu
